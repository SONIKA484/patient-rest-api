# Patient REST API

A simple REST API project built using PHP and MySQL for managing patient records.

## Features

- Get All Patients
- Get Patient By ID
- Add New Patient
- Update Patient Details
- Delete Patient

## Technologies Used

- PHP
- MySQL
- REST API
- JSON
- Postman
- Apache Server (WAMP)

## API Endpoints

### Get All Patients

GET
/api/patients

### Get Patient By ID

GET
/api/patients/{id}

### Add Patient

POST
/api/patients

### Update Patient

PUT
/api/patients/{id}

### Delete Patient

DELETE
/api/patients/{id}

## Project Structure

- config
- controllers
- models
- middlewares
- api
- .htaccess

## Testing

All API endpoints were tested using Postman.
